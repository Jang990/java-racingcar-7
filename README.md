# java-racingcar-precourse

"어떻게 구현할 것인가" 보다 "무엇을 테스트할 것인가"를 생각해보고 테스트할 기능들을 리스트로 나열한다.

## 기능 구현 목록
- 기본 입출력
- 사용자 이름 입력
- 사용자 시도 횟수 입력
- 자동차 경주 게임 시작
- 가장 많이 전진한 사람들 확인 
- 자동차 전진
- 자동차 상황 출력
- 전진 조건 확인
